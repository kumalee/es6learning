日文输入法规则：

一、拗音：

１直接打连音
```
きょ　kyo　
```
２前面加"L" 直接打小字
```
ょゅゃ　lyo lyu lya
```
二、促音：重复紧跟字字音
```
まって  matte
```
三、を、ん
```
を  wo　ん  nn 
```
四、长音
```
あー（减法键，打数字键的－不会出来长音）
```
五、其他
```
特殊符号：打 kigou（きごう）往下选
```
```
颜文字：打 kao（かお）往下选
```
例句
```
ちよっと待ってください

chilyottomattekudasai
```
待って＝まって　打好 matte 往下选就有中文了
